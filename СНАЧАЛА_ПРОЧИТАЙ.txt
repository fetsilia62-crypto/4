СНАЧАЛА РАСПАКУЙ АРХИВ

Не загружай файлы на GitHub прямо из окна 7-Zip или WinRAR.

Правильно:
1. Правой кнопкой по ZIP → «Извлечь все».
2. Открыть обычную папку passport-contract.
3. Проверить путь:
   .github\workflows\build-windows.yml
4. Только после этого перетащить содержимое папки на GitHub.

Если build-windows.yml окажется в корне репозитория, Actions его не увидит.
